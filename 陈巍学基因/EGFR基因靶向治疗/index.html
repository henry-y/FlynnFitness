EGFR基因
靶向治疗

来源：BV1PJ411k7XA 【陈巍学基因】视频35：EGFR基因靶向治疗

  • EGFR基因的突变，会在很大程度上影响到多种靶向治疗的有效性
  • EGFR基因的突变，也是相关的肿瘤靶向治疗中，最主要的生物标志物

EGFR基因简介

  1. 表皮生长因子受体，Epidermal growth factor receptor
    1. 别名：ErbB1/HER-1
    2. EGFR是ErbB基因家族的成员
      1. EGFR/ErbB1/HER-1
      2. ErbB2/HER-2
      3. ErbB3
      4. ErbB4
  2. EGFR基因在调控细胞生长和肿瘤发生的过程中，都起着重要的作用

EGFR蛋白的正常结构与功能

  1. EGFR基因定位：7号染色体短臂
  2. EGFR蛋白质分子量170KDa，有20%是糖链
  3. EGFR蛋白是一个跨膜蛋白，定位在细胞膜上，细胞膜外的部分负责与各种配体相结合，细胞膜内的部分负责信号转导，在激活的条件下，有酪氨酸激酶活性
  4. EGFR先以单体形式存在，而且单体的EGFR是没有酪氨酸激酶活性的；许多配体能与EGFR单体结合
    • 表皮生长因子Epidermal Growth Factor, EGF
    • 转化生长因子ɑ Transforming Growth Factor ɑ， TGF-ɑ
    • amphiregulin
    • betacellulin
    • epigen
    • epiregulin
    • heparin-binding EGF-like growth factor，HB-EGF
    • neuregulin 2b
  5. EGFR与配体结合后，两个带有配体的EGFR就会相互结合，形成有激酶活性的二聚体
    1. EGFR二聚体的激酶活性发挥作用，先把自身磷酸化，自身磷酸化后的EGFR二聚体，酪氨酸激酶活性会进一步加强
    2. 接下来就会开始磷酸化下游信号通路的蛋白，并就此激活下游信号通路的RAS，MAPK，Src，STAT3/5，PLCg，PKC和PI3K这一系列的信号通路
  6. 通过激活下游的信号通路，EGFR可以促进细胞的增殖，分化和迁移
  7. 在没有配体的情况下，细胞膜表面的EGFR单体会被持续地内吞到细胞内
    1. EGFR在细胞表面的半衰期是30min；EGFR蛋白的半衰期是20小时
    2. 被内吞到细胞里面的EGFR，一部分被降解，另外一部分会被再次带回到细胞膜表面

EGFR基因在肿瘤中的变异

  1. EGFR在肿瘤中的变异，可分为数量变异和结构变异两种
  2. 数量变异：在许多种肿瘤中，都能发现EGFR的异常高表达
    1. 通常情况下，一个细胞上会有4✖️10^4到1✖️10^5个EGFR分子
    2. 在肿瘤细胞中，可以观察到高达每个细胞2✖️10^6个EGFR分子
    3. 当细胞膜上有高浓度的EGFR单体分子时，两个EGFR分子就更可能在没有配体的情况下，自行聚合成一个二聚体；这个二聚体可以自我激活，并且进一步激活下游的细胞信号通路
    4. EGFR基因异常高表达的可能原因
      1. EGFR基因发生异常的拷贝数增加
      2. 调控EGFR的某些调控因子发生了变异，而导致EGFR的表达量的增加
  3. EGFR的结构变异
    1. EGFR的酪氨酸激酶活性位点，主要位于第18号外显子，到第24号外显子之间，EGFR的激活突变和耐药图片也都集中在这个区域
    2. 其中，第19号外显子的缺失变异（19del）和（位于21号外显子的）Leu858Arg突变（L858R），都会导致EGFR不需要与配体结合，就能直接拥有持续的酪氨酸激酶活性
    3. 一个比喻：银行金库的看守人员
      1. 正常情况下，要有银行的行长带着钥匙（配体）来，才能打开金库门（酪氨酸激酶活性）
      2. 在肿瘤中，看守人员自己偷偷配了金库的钥匙（激活突变），所以才能在银行行长不在的时候，也能直接打开金库的门，这样金库（细胞生长）就失控了
    4. 据统计，黄种人的非小细胞肺癌NSCLC中，有30-405的病例会出现EGFR基因的突变；白种人NSCLC中，这个比例是10-20%
    5. 在EGFR变异中，45%是19del，40%是21L858R，吉非替尼和厄洛替尼这两种药物会对有19del或L858R的NSCLC有较好的疗效
    6. 在用吉非替尼，厄洛替尼，埃克替尼等药物治疗4-6个月的时候，肿瘤往往会发生耐药突变，其中最常见的是T790M突变，即第790号氨基酸残基从苏氨酸变成了甲硫氨酸
    7. 肿瘤有了T790M突变后，就会对第一代酪氨酸激酶抑制类药物（一代TKI）产生耐药

针对EGFR进行肿瘤靶向治疗的药物和相应的研发策略

  1. 目前，靶向EGFR的药物，有抗体类药物和酪氨酸激酶抑制剂（小分子药）这两类药物
  2. 抗体药物：抗体特异性地识别EGFR的配体结合域，来与配体竞争，从而阻断配体对EGFR的激活作用
    1. 西妥昔单抗Cetuxiumab，商品名：爱必妥Erbitux
      1. 是第一个得到批准的抗EGFR-单克隆抗体药物，比EGFR的天然配体EGF和TGF-ɑ有更强的，与EGFR结合的能力；是鼠人源化的单克隆抗体
      2. 但只被批准用于进行头颈部肿瘤和结直肠癌CRC的治疗
      3. 西妥昔单抗与EGFR结合后，不仅能阻断天然配体与EGF的结合，还能启动细胞对EGFR的内吞作用，这减少了细胞表面EGFR的数量，并且被内吞的EGFR还会被讲解掉
      4. 西妥昔单抗不仅能与野生型EGFR，wt-EGFR（人话：正常的，没有发生突变的EGFR）结合，对于那些产生了变异的EGFR，例如L858R，T790M变异的EGFR也有效，并且能抑制这些有mutant-EGFR的肿瘤细胞的生长
    2. 帕尼单抗Panitumumab
      1. 完全人源的单克隆抗体
      2. 与EGFR有很高的亲和力，能够阻断配体与EGFR的结合；并且能够诱导细胞内吞EGFR，从而阻断肿瘤细胞的生长
      3. 经FDA批准，帕尼单抗可以用于：wt-KRAS的mCRC的治疗（m-metastatic，转移性）
  3. 酪氨酸激酶抑制剂Tyrosine Kinase Inhibitors，TKIs
    1. TKIs的共同特点：与ATP竞争EGFR的酪氨酸激酶活性中心，进而抑制EGFR对下游生物信号通路的激活作用
    2. 吉非替尼，商品名易瑞沙IRESSA
      1. 主要用于治疗既往接受过化学治疗的，局部晚期，或转移性NSCLC
      2. 吉非替尼是EGFR的可逆抑制剂，即吉非替尼与EGFR的结合是非共价的
    3. 与吉非替尼相似的药物
    4. TKIs药物，在用药4-6个月之后，往往会出现肿瘤耐药问题，其中最常见的耐药原因是EGFR发生了T790M突变
    5. 阿斯利康的新药Tagrisso/AZD9291，可以用于有T790M突变的NSCLC，对有T790M突变的NSCLC有很强的抑制作用，同时对wt-EGFR的影响较小
  4. 抗EGFR药物的常见副作用
    • 皮疹
    • 瘙痒
    • 腹泻
    • 肠道溃疡
    • 原因：EGFR是促进上皮细胞生长的重要促进因子，当EGFR被抑制，上皮细胞的生长就会被抑制，上述表现的原因是上皮细胞的正常更新受到了药物的抑制
  5. 针对EGFR的其他新药设计思路
    1. 抗体耦联化疗药物ADCs：Kadcyla用于HER-2阳性的，晚期乳腺癌的治疗，Kadcyla是曲妥珠单抗和Emtansine的耦联物
      • 曲妥珠单抗是抗HER-2蛋白的，而EGFR和HER-2同属于ErbB这个基因家族
      • 所以，可以借用针对HER-2的靶向药物设计策略，来设计针对EGFR的靶向药物
    2. 天然化合物的人工修饰
      1. 从各种已知的，与EGFR有亲和力的天然化合物开始，设计出能够抑制EGFR活性的衍生化合物
      2. 举例：紫草素
      3. 举例：金黄色葡萄球菌的蛋白A的Z功能域
      4. 举例：从抗EGFR抗体中，找出负责识别EGFR的那部分胎段（V区）
    3. 反义寡核苷酸，干扰RNAs：从转录、翻译的水平上，抑制EGFR的表达

针对EGFR靶向治疗当中所遇到的耐药问题

  1. 为什么抑制EGFR治疗有效：抑制了EGFR下游的MAPK和PI3K/Akt信号通路的活性
  2. 如果下游的MAPK，PI3K/Akt等信号环节发生了持续性的激活变异，就会绕开EGFR抑制剂的抑制作用，直接刺激细胞的增殖
  3. KRAS突变是肺腺癌中很重要的，对吉非替尼和厄洛替尼产生耐药的原因
  4. 另外，肿瘤细胞还能通过EGF和TGF-ɑ对VEGF的上调作用促进血管生成，也可以得到对EGFR抑制剂的抗性
  5. EGFR的内吞过程和降解过程发展异常，导致细胞膜表面的EGFR单体增多，大量的EGFR单体进一步与HER-2或HER-3的单体亚基形成异源二聚体，并激活后者，这会让NSCLC和头颈部鳞癌获得对EGFR抑制剂药物的耐药性
  6. EGFR基因拷贝数的异常增多，和干细胞生长因子活性的增强，也是NSCLC对西妥昔单抗产生耐药的一个原因
  7. EGFR的T790M突变，甲硫氨酸的空间位阻作用会让吉非替尼，厄洛替尼，埃克替尼等药物无法靠近酪氨酸激酶的活性中心，这是肿瘤对TKI耐药的最常见原因，其他的T854A、L747S、D761Y突变，也都因为同样的原因会引起对TKI的耐药

检测EGFR基因变异的临床工作方法

  1. 目前的手段：实时荧光定量PCR：Real-Time PCR/quantativePCR qPCR
  2. 未来可能的方法：数字液滴PCR，二代测序等
