训练时能不能喝可乐/雪碧？

碳水化合物在人体中的作用

  1. 提供人体50%-70%的能量供应
  2. 糖代谢的途径
    1. 糖的来源和加工：食物中的复杂的糖（多糖）经过消化道各种细胞的作用，分解为最简单的糖（单糖）
    2. 糖的去路
      1. 进入肝脏，合成肝糖原；肝糖原负责维持血糖的稳定
      2. 进入肌肉，合成肌糖原，这部分糖不能重新进入血液，只能用于给肌肉活动供能
      3. 极少部分经尿液排出
      4. 转化为脂肪
    3. 糖的主要作用
      1. 神经细胞和红细胞的直接能量来源
      2. 肌肉运动的能量来源
      3. 脂肪消耗的引物：减脂千万不要不吃碳水
  3. 缺少糖对身体的影响
    1. 两小时的高强度有氧运动，就足以耗尽肝脏和活动肌肉中的所有糖原，身体会逐渐进入疲劳状态
    2. 对于无氧和力量训练：身体在短时间内对能量需求增加，会促进糖进入产能效率更低的无氧代谢，糖原消耗的速度更快，身体会更早地疲劳
  4. 结论：为了维持训练时的糖原状态和运动表现，健身的时候喝可乐，雪碧这种所谓的“垃圾食品”是完全没有问题，而且是非常好的
    1. 需要担心的问题：可乐/雪碧含气，可能会成为在深蹲/硬拉等大重量训练时，腹内压已经很高的情况下，导致呕吐的“最后一根稻草”

训练期间的糖摄取建议

  1. 要想好好得增肌，甚至哪怕是从运动中恢复，碳水化合物都是不可或缺的，别再因为减脂而不吃碳水。
  2. NSCA（美国国家体能学会）的营养推荐：
    1. 抗阻训练期间，喝的饮料里需要有6-8%的碳水化合物，市面上一般的含糖饮料一般都处于这个范围内，选自己喜欢的就好
    2. 每个小时补充30-60g碳水，也就是一瓶左右的量（500ml），上述的量是建立在正常饮食（5-8g碳水/天）的基础上去计算的
  3. 摄入碳水的种类
    1. 运动前（前4小时）和运动时：首选升血糖快的，比如运动饮料，香蕉或其他你能找到的食物（但不要吃太咸的），糖分摄入量控制在30-40g
    2. 运动后30分钟-2小时：30-40g快速碳水+25g左右的优质蛋白
    3. 日常饮食：升糖慢的食物，比如糙米和全麦更加推荐
