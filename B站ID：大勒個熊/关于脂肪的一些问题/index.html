为什么运动中主要消耗碳水而不是脂肪

答：这样更有效率

  1. 比较糖和脂肪的分子式，糖的分子中含有氧原子的数量更多，对氧气的需要更少，在运动时氧气的供应量不可能无限大的前提下，使用糖作为主要的能量来源是更加“经济”的形式
  2. 氧气供应减少时，机体甚至会切换到主要进行无氧酵解的方式来产生能量，这种状态下对糖的消耗就更大了
  3. 总结：运动强度越大，身体越倾向于用碳水来供能，脂肪供能的比例就越小；反过来也是这样，所以身体最安静的时候，也就是睡觉的时候，脂肪供能的比例最高，接近70%

有氧半小时后才开始消耗脂肪，是真的吗？

答：显然不是

  1. 不管是运动还是睡觉的时候，脂肪组织都会分解，形成游离的脂肪酸的
  2. 靠有氧运动减脂，是在供能比例和脂肪的总消耗量两者的关系上去寻求一个平衡点；一个合理的供能比例✖️较长的运动时间，才能让脂肪消耗量足够多

高强度间歇循环训练HIIT为什么能减脂？

  1. 做HIIT的半小时到一小时里，身体总的热量消耗是很大的，真的很辛苦
  2. 后续的生理过程需要身体持续消耗脂肪，提供能量
    1. 在HIIT中，身体吸入氧气不足，产生了所谓的“氧债”，这时的碳水利用是不充分的
    2. 无氧呼吸大量产生乳酸，很有可能参与了运动过程中酸胀感的产生，运动结束后身体需要排出乳酸和其他代谢废物
    3. 运动造成的肌肉围观损伤也需要修补
  3. 总结：HIIT消耗的脂肪有很大一部分不在运动过程里，而是在运动过程后；力量抗阻训练也是一样的道理
  4. 补充：HIIT对身体素质的基础要求比较高（因为有很多跳跃类的动作），不建议没有运动习惯的朋友们一上来就挑战这么高难度的项目

减脂不吃碳水是好方法吗？

  1. 糖的主要作用
    1. 神经细胞和红细胞的直接能量来源
    2. 肌肉运动的能量来源
    3. 脂肪消耗的引物：减脂千万不要不吃碳水
  2. 训练强度越高，身体对糖的需求就越高；糖摄入不足时，肌糖原的储备会急剧下降，运动表现也会大打折扣的
  3. 对于没有训练习惯的朋友
    1. 碳水不够会影响神经细胞的活性，人会觉得很容易累
    2. 脂肪消耗的生理过程也会受到影响
  4. 推荐的做法
    1. 正常吃饭
    2. 少盐少油
    3. 细嚼慢咽
    4. 米饭搭配升糖指数低的主食：如南瓜，玉米
    5. 多吃青菜
