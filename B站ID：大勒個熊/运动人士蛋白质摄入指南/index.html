一个简单版本的蛋白质代谢

  1. 蛋白质的来源：食物中，每日约80-150g
    1. 1个鸡蛋：6g
    2. 一个鸡大腿：16g（去皮/完整都一样）
  2. 蛋白质的处理
    1. 小肠吸收，分解为22种氨基酸（包含8种人体不能自主合成的必需氨基酸），经血液循环运送到肝
    2. 肝脏对蛋白质的处理
      1. 合成代谢：合成肌肉，激素，酶等人自己需要的蛋白质，参与生理功能
      2. 分解代谢：氨基酸脱去氨基，形成ɑ-酮酸，被氧化燃烧，重新合成氨基酸或辅助糖和脂肪的代谢
      3. 经过体液流失
  3. “我吃很多瘦肉来减肥”有用吗？
    1. 没有。
    2. 对于蛋白质来说，超出满足合成代谢需求的蛋白质，在身体里会转化为糖和脂肪存储起来
  4. “传统手艺”会导致严重的蛋白质流失吗？
    1. 不会。每100ml乳白色液体里只有不到5g蛋白质，而且每次传统手艺消耗的乳白色液体只有不到10ml，所以相比于每日摄入的蛋白质微乎其微。
    2. 传统手艺后立刻补充营养快线一瓶：500ml营养快线中含有5g蛋白质和总计227大卡的热量，蛋白质是补充回来了，更重要的是实打实地积攒了脂肪hhh

肌肉是如何利用蛋白质的

  1. 运动后，肌肉纤维（由蛋白质组成）出现了微小的损伤，由此肌肉细胞里蛋白质合成的需要超过了蛋白质分解的需要
  2. 肝细胞会囤积一定的氨基酸，称为氨基酸池；运动后氨基酸池开始发挥作用，释放游离氨基酸进入血液，提供给肌细胞，用来更新肌肉纤维所需要的蛋白质

关于支链氨基酸

  1. 支链氨基酸(BCAA, Branch-Chain Amino Acids)的作用：
    1. 在运动时，被骨骼肌氧化，提供能量
    2. 促进运动之后肌肉蛋白质的合成
    3. 减少运动导致的肌肉损伤
  2. 什么样的人需要BCAA：
    1. 支链氨基酸的效果，在进行长时间有氧运动，比如长跑和自行车的人群中才比较明显；健身房训练者由于组间休息的存在，食物中的BCAA就足够了
    2. 减脂期人士：减脂期人士由于热量缺口很大，食物中的BCAA可能不足以维持减脂期力量训练+有氧的需要，可以推荐补充
    3. 土豪随意，但这玩意根本不好喝
    4. 别扯“肝性脑病”：这玩意确实是医学院学生内科学的重点，但会得肝性脑病的人一般都得躺床上，没什么能力进行运动

如何选择合适的蛋白质？

  1. 定量分析：蛋白质消化率校正氨基酸分数（PDCAAS，Protein Digestibility-Corrected Amino Acid Score），PDCAAS的最高值为1，且数值越高代表蛋白质质量越好
  2. 常见的蛋白质的PDCAAS值
  3. 牛奶蛋白与蛋白粉
    1. 牛奶蛋白=乳清蛋白（占比80%）+酪蛋白（20%）
    2. 乳清蛋白和酪蛋白的区别
      1. 酪蛋白的消化速度更慢，有更加强的抵抗肌肉分解的功效
      2. 乳清蛋白更加能促进血液氨基酸水平快速增加，促进蛋白质的合成
    3. 两者结合起来补充，效果更好哦
    4. 蛋白粉是牛奶蛋白的提取物，市面上不同品牌的蛋白粉成分没什么区别，按照喜欢的口味和能够承受的价格选就好
    5. 补充摄入蛋白粉的前提：从常见的天然食物里尽可能吃够干净的蛋白质，还是推荐自己做饭；蛋白粉不能当饭吃，不要舍本逐末
